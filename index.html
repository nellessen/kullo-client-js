Kullo Messaging Client

Kullo provides secure and easy-to-use messaging. You can use it as an alternative to E-Mail, chat and other (instant) messengers if data security is important to you.

Learn more »

STRONG ENCRYPTION

Kullo relies on a strong public key algorithm from the open source software openSSL that is widely spread and actively developed.

View details »

FREE AND OPEN

It's an open infrastructure based on established web standards. There is no central company that controls all the communication. Everyone can become a messaging provider. Furthermore we want to have Kullo on every device possible and encourage the best app developers to build client software for it.

View details »

HUMAN AND FUN

Besides all that it is human and fun to use. Everything is based on your Kullo address and your password only. No key management, no configuration, no nothing.
